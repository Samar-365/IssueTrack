Project Title: Issue Tracking System (Mini Jira)
Version 1.0
1. Introduction
1.1 Purpose

The purpose of this project is to develop a web-based Issue Tracking System that helps organizations manage projects, track issues, assign tasks, and monitor progress efficiently. The system enables administrators, managers, and employees to collaborate through a centralized platform.

1.2 Scope

The Issue Tracking System allows users to:

Create and manage projects
Create, assign, and track issues/tickets
Monitor task status
Set priorities and deadlines
Add comments and updates
Generate reports
Maintain activity logs

The system will provide role-based access control for Admins, Managers, and Employees.

1.3 Objectives
Improve project tracking
Simplify issue management
Enhance team collaboration
Maintain issue history and accountability
Provide project progress visibility
2. Overall Description
2.1 Product Perspective

The system is a standalone web application that stores all project and issue information in a MySQL database.
